## Overview

The `AudioWaveform` component provides a visual representation of audio activity through animated bars. It creates a simple but effective waveform animation to indicate when audio is being recorded or processed.

## Features

- **Animated Bars**: Five animated bars with staggered timing for natural wave effect
- **Active State**: Smooth transition between active and inactive states
- **Lightweight**: Minimal DOM elements and CSS animations
- **Customizable**: Supports custom styling via className prop
- **Theme Compatible**: Works with light and dark themes using `bg-current`

## Props

<ParamField path="active" type="boolean" default="false">
  Controls whether the waveform animation is active. When `true`, bars animate with wave motion. When `false`, bars shrink to minimal height.
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes for custom styling of the container element
</ParamField>

## Animation Details

### Active State

When `active={true}`:
- Each bar animates vertically using the `animate-music-bar` animation
- Staggered delays create a wave effect (0.1s, 0.2s, 0.3s, 0.4s, 0.5s)
- Bars vary in height naturally during animation
- Full opacity for clear visibility

### Inactive State

When `active={false}`:
- Bars shrink to 4px height
- Opacity reduced to 20%
- Smooth 300ms transition between states

### CSS Animation

The component relies on a CSS keyframe animation defined in your global styles:

```css
@keyframes music-bar {
  0%, 100% {
    height: 4px;
  }
  50% {
    height: 16px;
  }
}

.animate-music-bar {
  animation: music-bar 0.8s ease-in-out infinite;
}
```

## Performance

- **Lightweight**: Only 5 DOM elements
- **CSS Animations**: Uses GPU-accelerated CSS animations
- **No JavaScript Animation**: Pure CSS for smooth 60fps animation
- **Conditional Rendering**: Can be conditionally rendered to save resources

## Browser Support

The component uses:
- CSS `animation` (widely supported)
- Flexbox layout (all modern browsers)
- `bg-current` color (CSS Variables)

Works in all modern browsers. Gracefully degrades in older browsers (shows static bars).

## Variants

### Pulse Style

```tsx
function PulseWaveform({ active }) {
  return (
    <div className="flex items-center gap-0.5 h-4">
      {[1, 2, 3].map((i) => (

          className={cn(
            'w-2 h-2 rounded-full bg-current',

```
