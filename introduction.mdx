# Welcome to Gima AI Chatbot

Gima is an intelligent assistant designed specifically for asset and maintenance management at UNEG. Built with cutting-edge AI technology, it provides a conversational interface for managing work orders, analyzing equipment, and streamlining maintenance operations.

## Technology Stack

Gima is built with modern web technologies:

- **Next.js 16** - App Router with React Server Components
- **React 19** - Latest React with concurrent features
- **Vercel AI SDK v5** - Multi-provider AI integration
- **TypeScript 5** - Type-safe development with strict mode
- **Tailwind CSS 4** - Modern utility-first styling
- **GROQ & Google Gemini** - Multiple AI model support

## Use Cases

<AccordionGroup>
  <Accordion title="Equipment Maintenance">
    Query equipment status, view maintenance history, and create work orders using natural language commands like "Show me all HVAC units in sector 3" or "Create urgent maintenance for BCA pump."
  </Accordion>
  <Accordion title="Inventory Management">
    Check spare parts availability, track inventory levels, and identify low-stock items with queries like "Which parts are running low?" or "Show me the inventory for air conditioning components."
  </Accordion>
  <Accordion title="Visual Inspection">
    Upload images of equipment or parts for AI-powered analysis using Gemini Vision. The system can identify components, detect issues, and suggest maintenance actions.
  </Accordion>
  <Accordion title="Documentation Processing">
    Upload maintenance manuals and technical documents as PDFs. Gima extracts content and answers questions about procedures, specifications, and troubleshooting steps.
  </Accordion>
</AccordionGroup>
